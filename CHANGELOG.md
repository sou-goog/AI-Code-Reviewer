# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-09

### Added
- CLI tool with multiple review modes (staged, uncommitted, last-commit)
- Google Gemini AI integration for code analysis
- Severity categorization (Critical, Warning, Suggestion, Positive)
- Multiple output formats (terminal, markdown, JSON)
- Web dashboard with Streamlit
- GitHub Actions workflow for automatic PR reviews
- Pre-commit hooks for local code review
- Custom rules engine with YAML configuration
- Configuration file support (.codereview.yaml)
- Professional documentation and guides
- Security policy
- Contributing guidelines
- Issue and PR templates

### Features
- Pattern-based custom rules
- Ignore patterns for files/directories
- Rich terminal UI with markdown rendering
- Beautiful gradient-based web interface
- Real-time code analysis
- Downloadable review reports

### Tech Stack
- Python 3.9+
- Google Generative AI (Gemini 2.5 Flash)
- Streamlit
- GitPython
- Typer
- Rich
- PyYAML
